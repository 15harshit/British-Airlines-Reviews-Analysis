# ✈️ British Airways Customer Reviews Analysis Dashboard

An end-to-end data analytics project built using Python and Tableau to analyze British Airways customer reviews across aircraft types, seat classes, traveller segments, and countries.

The project analyzes a dataset containing 1,324 customer reviews (2016–2023) and transforms raw, scraped review data into meaningful business insights through KPI development, interactive visualizations, and an executive-level dashboard.

### Key Outcomes

- Analyzed overall customer satisfaction and rating trends over time.
- Identified top- and bottom-performing aircraft by average rating.
- Compared satisfaction across traveller types and seat classes.
- Evaluated which service categories most strongly drive recommendations.
- Designed an interactive dashboard for business decision-making.

## 📌 Project Overview

This project presents an end-to-end Customer Review Analytics solution developed using Python and Tableau Public for the British Airways Customer Reviews dataset.

The project follows a complete analytics workflow including data cleaning, KPI development, satisfaction analysis, trend visualization, and dashboard creation to generate actionable business insights.

The final dashboard provides an executive-level overview of customer satisfaction, service category performance, aircraft comparisons, traveller segment analysis, and global review geography.

## 📈 Key Business Insights

- Analyzed 1,324 customer reviews with an average overall rating of **4.19 out of 5**.
- Only **43.2%** of reviewers said they would recommend British Airways, despite a relatively high average rating.
- Satisfaction peaked in **2020 (4.84 avg)** and declined sharply to **2.61 by 2023**.
- The **Boeing 747-400** emerged as the highest-rated aircraft, averaging **4.7** across 97 reviews.
- **Business Class** reviewers reported lower satisfaction (**3.93**) than **Economy Class** reviewers (**4.36**).
- **Family Leisure** travellers recorded the lowest average rating among all traveller segments.
- **Value for Money** showed the strongest correlation with recommendation likelihood (**r = 0.78**).

## 📁 Dataset

### British Airways Customer Reviews Dataset

The dataset contains information including:

- Author
- Review Date
- Reviewer Country
- Aircraft
- Traveller Type
- Seat Type
- Route
- Date Flown
- Recommended (Yes/No)
- Trip Verified Status
- Overall Rating
- Seat Comfort
- Cabin Staff Service
- Food & Beverages
- Ground Service
- Value for Money
- Entertainment

A secondary Countries dataset (Country, Code, Continent, Region) was joined to the reviews data to enable continent-level analysis.
